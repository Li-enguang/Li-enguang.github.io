----------

## 使用

#### 1、安装

```bash
cd ./themes/
git clone https://github.com/JoeyBling/hexo-theme-yilia-plus.git ./yilia-plus
```

#### 2、配置

修改hexo根目录下的 `_config.yml`  ➡️  `theme: yilia-plus`

#### 3、更新

```bash
cd themes/yilia-plus
git pull
```

## 四、配置

&#160;&#160;&#160;&#160;主题配置文件在主目录下的`_config.yml`，请根据自己需要修改使用。
